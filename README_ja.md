# Helpdesk

## このフォークについて

このリポジトリは、Moodle 用プラグイン Kopere Helpdesk のフォークです。

互換性を保つため、Moodleコンポーネント名は `local_helpdesk` のまま維持しています。そのうえで、Kopere Dashboard / Kopere BI / GeniAI への依存を外し、Moodle標準機能だけで動く問い合わせチケットシステムとして改修しています。

## 概要

このプラグインは、Moodle上で問い合わせを受け付け、管理者またはカテゴリ担当者が返信・対応できるようにするためのローカルプラグインです。

ユーザは問い合わせチケットを作成し、担当者はカテゴリごとの一覧やチケット詳細画面から返信できます。返信や新規チケット作成時には、Moodle標準のメッセージ通知を利用します。

## 主な機能

- ユーザによる問い合わせチケット作成
- チケット一覧・詳細表示
- カテゴリによる問い合わせ分類
- カテゴリごとの対応ユーザ割り当て
- 管理者または担当者による返信
- ステータス・優先度・カテゴリの変更
- 添付ファイル
- Moodle標準メッセージ通知
- ナレッジベース記事
- CSVエクスポート

## 動作要件

この独立版では、以下のプラグインは不要です。

- Kopere Dashboard
- Kopere BI
- GeniAI

Moodle本体のみで動作することを目標にしています。

## 現在の状態

Phase 1 として、Kopere / GeniAI 依存を外した初期版です。

確認済みのこと:

- Moodleへのインストール
- チケット作成
- チケット返信
- 返信内容の表示
- ステータス変更
- Moodle Web通知

今後確認・改善したいこと:

- メール通知
- 添付ファイルのアクセス制御
- 担当者割り当て機能の強化
- 内部メモ
- 活動ログ
- 同時返信を防ぐための競合チェック
- ユーザ・コース選択UIの改善
- 日本語言語ファイルの追加

## 元プラグインから外した機能

このフォークでは、以下の機能を意図的に外しています。

- Kopere BI によるレポート画面
- GeniAI によるAI返信生成
- Kopere Dashboard に依存した通知テンプレート処理
- Kopere Dashboard に依存したユーザ・コース検索UI

これらは、必要に応じて今後 Moodle標準APIを使った形で再設計する予定です。
